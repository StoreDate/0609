# ОБОД — Образная Обработка Данных

**ОБОД** — это диагностическая система, объединяющая нумерологию, психосоматику и образное мышление в едином космо-OBD интерфейсе.

> «Считываем показатели души через приборную панель сознания»

## ✨ Возможности

- 🚀 ECU ID — число судьбы (нумерологический анализ)
- 🔑 Активационный ключ — число души
- 🎛️ Диагностика 4 контуров: Двигатель, Память, Впрыск, Топливо (стихии)
- 📊 Мандала — энергетический профиль личности
- 🤼 Сопоставимость — совместимость по числам
- 🧠 Три психологических теста: ТОБОЛ, ВКБ, Алекситимия
- 📖 Бортовой журнал с развёрнутой расшифровкой
- 💊 Психосоматический профиль (зоны риска, практики)

### 🐍 Версия для разработчиков (исходный код)

**Python 3.12+**
- `core_data.py` — вся нумерологическая логика (2000+ строк)
- `text_generator.py` — генератор живых текстов
- `ui_obod_qt.py` — Qt интерфейс
- `main.py` — точка входа для локального запуска
- `run_obod.py` — умный запускатор
- `obod.html` — HTML интерфейс (для GAS)
